Part 1: Introduction & Setup (45 min)

Activity 1.1: Understanding Ucode (15 min)
Read and understand:

-   What is Ucode? (Backend as a Service)
-   Database introspection concept
-   How Ucode differs from traditional CMS

Answers:

-   Ucode is a platform that makes designing database patterns and keeping track of changes on your website easy and understandable for non-tech users
-   Database introspection is a concept of a system that can examine an already existing database and understand its structure(tables, columns, data types, etc..)
-   Ucode is a speacial CMS that is created and managed by the Udevs team. Its is a platform where Udevs developers manage websites and database systems

Activity 1.3: Explore the Interface (15 min)
Navigate through each section and document in notes.md:

-   Menu: What can you do here?
-   Tables: How many tables exist?
-   Items: What are items?
-   Settings: What settings are available?

Answers:

-   On the menu section we can create folders and files(tables, webpages and microfrontend) and navigate through all these folders and files
-   We can add as many tables as we want, there is no limitation stated
-   Items are like rows in a database. Each item has its data type and value. We can create items just by clicking the create item button
-   There are 3 categories of settings in the settings section: account, project management and advanced
